# 工具页面使用指南

## 快速开始

### 1. 启动项目

```bash
pnpm dev
```

### 2. 访问工具页面

浏览器打开：http://localhost:4321/tools

### 3. 使用工具

在工具页面中：

1. 点击导航栏的 **"Tools"** 链接
2. 选择要使用的工具页签
3. 在文本框中输入或粘贴要处理的文本
4. 点击相应的按钮进行操作

## 功能介绍

### 📝 字符串格式化（默认页签）

#### 支持的格式类型

- **JSON**: 美化 JSON 代码
- **XML**: 美化 XML 代码
- **Protobuf**: 美化 Protobuf 消息定义

#### 使用步骤

1. 选择格式类型（单选按钮）
2. （可选）在搜索框输入关键词，高亮匹配行
3. 在文本框输入内容
4. 点击 **"格式化"** 按钮
5. 点击 **"清空"** 按钮重置

#### 示例

**JSON 格式化**
输入：

```json
{ "name": "test", "value": 123, "items": ["a", "b"] }
```

输出：

```json
{
  "name": "test",
  "value": 123,
  "items": ["a", "b"]
}
```

**XML 格式化**
输入：

```xml
<root><child><name>test</name><value>123</value></child></root>
```

输出：

```xml
<root>
  <child>
    <name>test</name>
    <value>123</value>
  </child>
</root>
```

**Protobuf 格式化**
输入：

```protobuf
message Person {
required string name=1;
optional int32 id=2;
}
```

输出：

```protobuf
message Person {
  required string name = 1;
  optional int32 id = 2;
}
```

#### 搜索功能

- 在搜索框输入文字
- 自动过滤并高亮包含关键词的行
- 搜索不区分大小写
- 无输入时显示所有内容

### 常见问题

**Q: 格式化失败怎么办？**
A: 检查输入的文本格式是否正确。系统会提示具体的错误信息。

**Q: 如何清除搜索结果？**
A: 清空搜索框，或点击格式化按钮清空文本。

**Q: 支持批量处理吗？**
A: 当前版本需要逐个处理。未来版本将支持批量功能。
